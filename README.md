# WebServicesBUAP

Servicio web desarrollado con .NET Framework y WCF. Este sistema tiene la finalidad de explorar el funcionamiento de dichas tecnologías por lo que no tiene un proposito más allá de eso.

Este servicio web exponé un conjunto de funciones para gestionar una base de datos en tiempo real de Firebases.
